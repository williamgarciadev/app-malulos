# Quick Reference

## Arranque rapido

```bash
npm run seed
npm run dev:all
```

## URLs

- Frontend: http://localhost:5174
- API Health: http://localhost:3000/api/health

## Variables clave

```env
DATABASE_URL=postgresql://postgres:TU_PASSWORD@localhost:5432/malulos_pos
PORT=3000
VITE_API_URL=http://localhost:3000/api
```
